# House Price Analysis
A machine learning project that analyzes house prices based on different property features such as number of rooms, parking, and elevator availability. It uses XGBoost for feature importance and Matplotlib for visualization.

House Price Analysis

This project analyzes housing data and visualizes the impact of different property features on house prices.

The analysis is performed using XGBoost for feature importance and Matplotlib for data visualization.

Features

The project can analyze features such as:

* Number of bedrooms
* Parking availability
* Elevator availability
* Property area
* Building age
* Any other features available in the dataset

Technologies Used

* Python
* Pandas
* XGBoost
* Matplotlib

Dataset

The original dataset used during development is not included in this repository.

To use this project, place your own housing dataset in the project directory and update the dataset path in the source code if necessary.

The dataset should contain a target column representing the house price and additional columns describing property features.

Installation

pip install -r requirements.txt

Usage

python main.py

After running the project, charts will be generated showing the relationship between house prices and the available property features.

Purpose

This project was created to practice machine learning, feature importance analysis, and data visualization using real estate data.
